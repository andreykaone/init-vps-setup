# VPS Initial Setup Script

Интерактивный скрипт для быстрой и безопасной настройки нового сервера (Ubuntu/Debian).

## Что делает этот скрипт:
1.  **Обновление**: Обновляет системные пакеты.
2.  **Пользователь**: Создает нового пользователя с правами `sudo`.
3.  **SSH Security**:
    *   Устанавливает ваш SSH-ключ.
    *   Меняет стандартный порт SSH (22 -> ваш выбор).
    *   Отключает вход под `root` и аутентификацию по паролю.
4.  **Firewall (UFW)**: Закрывает все порты, кроме SSH и Web (80, 443).
5.  **Fail2Ban**: Защищает от брутфорс-атак.
6.  **Swap**: Создает файл подкачки для стабильности системы.
7.  **Timezone**: Настраивает системное время.
8.  **Auto-updates**: Включает автоматические обновления безопасности.
9.  **Docker**: (Опционально) Устанавливает Docker Engine и Docker Compose, добавляет пользователя в группу `docker`.

## Как запустить на новом сервере:

Зайдите на ваш VPS под root и выполните:

```bash
curl -sSL https://raw.githubusercontent.com/andreykaone/init-vps-setup/main/init-vps-setup.sh -o setup.sh && chmod +x setup.sh && sudo ./setup.sh
```

## Важные замечания:
*   Скрипт предназначен для **чистых** систем (Ubuntu 20.04/22.04+ или Debian).
*   **Внимание**: Не закрывайте текущую SSH сессию, пока не убедитесь, что можете войти под новым пользователем по новому порту!
